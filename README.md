# ConductorJS

## What is ConductorJS?
```
console.log("A JavaScript website to learn JavaScript")
```
This website allows Computer Science students (and others!) to learn JavaScript by providing easy to understand, 
<br>
comprehensible JavaScript-related information.
Examples will also be provided for further understanding!

## Frameworks and Tools involved
- React (Frontend)
- NodeJS & ExpressJS (Backend)
- Postgresql (Database)
- Figma (UI design)
